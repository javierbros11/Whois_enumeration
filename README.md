
# WHOIS Enumeration

Se trata de un programa que permite consultar la base de datos Whois para mostrar por pantalla toda la información obtenida en base a un nombre de dominio que proporcionemos.
## Instalación

Para el correcto funcionamiento del script requerimos instalar previamente Python en nuestro equipo.

Instalar Python en caso de que no lo tenga:

[Download Python](https://www.python.org/downloads/)

Adicionalmente requerimos instalar el paquete de Whois para Python (documentado en el fichero **requirements.txt**):

```bash
  pip install python-whois
```



## Funcionalidades

- **Información**: El script devolverá datos otorgados por Whois que se pueden consultar de manera pública, es información que normalmente es solitada a las empresas con el fin de poder crear un dominio en Internet.

- **Fácil y rápido**: Al ejecutar el script únicamente nos solicitará que introduzcamos un nombre de dominio a buscar. A continuación devolverá toda la información obtenida mostrada en la terminal.
